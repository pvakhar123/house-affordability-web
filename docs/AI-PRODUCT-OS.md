## KEY FRAMEWORKS REFERENCE

### Strategy
- **Wardley Mapping** — Map capabilities from value to commodity. Find where to differentiate.
- **Playing to Win** — Where to play, how to win, what capabilities needed.
- **Working Backwards** — Write the press release before building. Forces clarity.
- **7 Powers** — Scale, network effects, counter-positioning, switching costs, brand, cornered resource, process power.
- **Blue Ocean Strategy** — Eliminate/Reduce/Raise/Create to find uncontested space.
- **Three Horizons** — 70% core, 20% adjacent, 10% breakthrough.

### Prioritization
- **RICE** — (Reach x Impact x Confidence) / Effort
- **ICE** — Impact x Confidence x Ease (1-10 each)
- **Kano Model** — Basic/Performance/Delighter categorization
- **MoSCoW** — Must/Should/Could/Won't
- **Cost of Delay** — Value ÷ Time. Sequence by urgency.
- **LNO** — Leverage (70%) / Neutral (20%) / Overhead (10%)
- **Eisenhower Matrix** — Important+Urgent grid for saying no.

### Discovery & Research
- **Continuous Discovery** — 2-4 customer calls/week, every week.
- **Opportunity Solution Trees** — Outcome → Opportunities → Solutions → Experiments
- **Double Diamond** — Discover (diverge/converge) → Develop (diverge/converge)
- **Design Sprint** — 5-day: Map → Sketch → Decide → Prototype → Test
- **Assumption Mapping** — Importance x Evidence. Test high-risk first.
- **JTBD Canvas** — Functional + emotional + social dimensions of the job.
- **Empathy Mapping** — Says/Thinks/Does/Feels + Pains/Gains

### Growth & Metrics
- **North Star Metric** — The one metric that captures core value delivery.
- **Pirate Metrics (AARRR)** — Acquisition → Activation → Retention → Revenue → Referral
- **Growth Loops** — Self-reinforcing: user action → new users → more actions
- **Sean Ellis PMF Test** — "Very disappointed" > 40% = product-market fit

### Execution
- **Shape Up** — 6-week cycles, appetite-based scoping, cool-down periods
- **GIST Planning** — Goals → Ideas → Steps → Tasks
- **Now/Next/Later Roadmap** — Time-based without false precision
