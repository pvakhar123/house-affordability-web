# HomeIQ — AI Home Affordability Calculator

An AI-powered home affordability analysis platform that helps buyers make informed decisions. Built with Next.js, Claude AI, and real-time market data.

**Live:** [aicalculator.homes](https://aicalculator.homes)

## Features

- **Affordability Analysis** — Income, debts, down payment, and budget simulation
- **Live Market Data** — Real-time mortgage rates from the Federal Reserve, median prices, and trends
- **Area Intelligence** — Interactive Google Map, commute calculator, and nearby amenities
- **Matching Properties** — Real listings from Realtor.com within your budget
- **Rent vs Buy** — Year-by-year financial comparison
- **Loan Programs** — FHA, VA, conventional, and more with personalized recommendations
- **Closing Costs** — State-specific estimates
- **Risk Assessment** — Stress tests, DTI analysis, and market risk scoring
- **Equity Projections** — 5+ year amortization and equity buildup charts
- **AI Summary** — Detailed, personalized report generated by Claude
- **Interactive Chat** — Ask follow-up "what if" questions about your analysis

## Tech Stack

- **Framework:** Next.js 16, React 19, TypeScript, Tailwind CSS 4
- **AI:** Anthropic Claude (via @anthropic-ai/sdk)
- **Database:** Neon Postgres (via Drizzle ORM)
- **Auth:** NextAuth v5 + Google OAuth
- **Maps:** Google Maps Platform + Mapbox geocoding
- **Payments:** Stripe
- **Charts:** Recharts
- **Monitoring:** Sentry, PostHog, Langfuse

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Create a `.env.local` file:

```env
# Required
ANTHROPIC_API_KEY=
FRED_API_KEY=

# Database & Auth
POSTGRES_URL=
AUTH_SECRET=
AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=

# Maps
MAPBOX_ACCESS_TOKEN=
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
GOOGLE_MAPS_API_KEY=

# Payments
STRIPE_SECRET_KEY=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=

# Analytics & Monitoring (optional)
LANGFUSE_PUBLIC_KEY=
LANGFUSE_SECRET_KEY=
NEXT_PUBLIC_POSTHOG_KEY=
NEXT_PUBLIC_POSTHOG_HOST=
SENTRY_AUTH_TOKEN=

# Data APIs (optional)
RAPIDAPI_KEY=
BLS_API_KEY=
```

## Deployment

Deployed on Vercel. Push to `main` for production, `staging` for preview.

```bash
npx vercel --prod    # production → aicalculator.homes
npx vercel           # staging preview
```
